# Esport League Manager (WIP)

Ce projet est un travail étudiant de gestion de ligue E-sport. L'objectif est de créer une application console permettant de gérer des joueurs, leurs statistiques et leurs équipes.

# État du projet

Le projet est actuellement en cours de développement. La partie concernant les joueurs est fonctionnelle, la gestion des équipes est la prochaine étape.

# Architecture

Le code est structuré selon le modèle MVC (Modèle-Vue-Contrôleur) avec une couche DAO pour la communication avec la base de données SQL. L'idée est de garder un code propre et bien séparé dès le début.

# Fonctionnalités actuelles

## Joueurs : Ajout, affichage de la liste, modification et suppression.

## Stats : Calcul automatique du ratio K/D dans le modèle.

## Base de données : Connexion via JDBC avec requêtes sécurisées.

# À venir
Création et gestion des équipes.

Liaison entre un joueur et son équipe (clés étrangères).

Historique des matchs.

Cloner le dépôt.
